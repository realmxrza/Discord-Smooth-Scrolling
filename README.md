This script is made specifically for mouse scroll wheels. Make scrolling feels less snappy

( TEMPORARY )
Press Ctrl + Shift + I
Type the following exactly as shown "allow pasting" ( for first time user) and press Enter on the Console tab
Paste the smoothscroll.js into the Console tab
Hit Enter

( VENCORD PRELOAD )
Press win + R
Navigate to " %appdata%\Vencord "
Open "dist" folder
Download smoothscroll.js and paste the file in the folder
Open "preload.js" with Notepad
Paste this on the bottom " require("./smoothscroll.js"); "
Restart Discord
